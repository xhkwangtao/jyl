# 九眼楼网页轨迹编辑器

这是一个本地运行的网页工具，用来直接读取 `KMZ` 中的轨迹、标注点和照片，方便你边看图边给景点命名，边拖点边调整路线。

## 启动

```bash
cd editor
npm install
npm run dev
```

构建产物：

```bash
npm run build
```

样本校验：

```bash
npm run verify:sample
```

## 当前能力

- 导入任意 `KMZ`
- 解析轨迹、照片标注点和文本标注点
- 在网页里拖动景点位置
- 在网页里拖动轨迹点、插入轨迹点、删除轨迹点
- 预览每个点位的照片
- 编辑景点名称、类型、描述、展示方式
- 导出 `JSON`
- 导出 `module.exports = ...` 的 JS 包装文件
- 导出包含图片和 JSON 的 ZIP 包

## 说明

- 当前编辑器的空间视图已经使用 `GCJ-02` 坐标，导出结果可以继续对接微信小程序地图。
- 仓库里没有现成的腾讯网页地图 `Web Key`，所以首版没有把可用性绑定在在线底图上。
- 如果你只是为了整理景点名称、调整点位和导出小程序数据，这个版本已经可以直接用。
